# [Catalina]

Boilerplate para empezar cualquier tipo de Web App. 
Gestionado con Gulp para manejar tareas, Bower para administrar librerias y Sass.

## Empezando

Instalar dependencias:

```bash
$ npm install
```

```bash
$ bower install
```

## Como usarlo?

### Tareas

```bash
$ gulp serve
```
